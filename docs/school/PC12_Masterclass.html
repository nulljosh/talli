Unit 1Sequences & Series

Arithmetic

aₙ = a₁ + (n − 1)d Sₙ = n/2 · (a₁ + aₙ) = n/2 · (2a₁ + (n−1)d)
  • d = common difference = a₂ − a₁ (constant gap)
  • Finding n: sub known values into aₙ formula, solve for n
  • Partial sum (terms i to j): Sⱼ − Sᵢ₋₁

Geometric

aₙ = a₁ · r^(n−1) Sₙ = a₁(1 − rⁿ) / (1 − r) [r ≠ 1] S∞ = a₁ / (1 − r) [only when |r| < 1]
  • r = common ratio = a₂/a₁
  • Finding n: aₙ = a₁rⁿ⁻¹ → take log both sides → n = 1 + log(aₙ/a₁)/log(r)
Convergent vs Divergent — most tested concept

Converges (S∞ exists) iff |r| < 1

Diverges (no S∞) iff |r| ≥ 1

r = −0.99 → |r| = 0.99 < 1 → CONVERGES. Negative r ≠ divergent.

r = 1.01 → |r| = 1.01 ≥ 1 → DIVERGES.

Sigma Notation

Σᵢ₌₁ⁿ (a₁ + (i−1)d) ← arithmetic sum to n terms Σᵢ₌₁ⁿ a₁·r^(i−1) ← geometric sum to n terms

Bottom of sigma = starting index; top = ending index; expression = general term.

Worked Example — Find S∞
Q: Σᵢ₌₁^∞ 8·(−0.5)^(i−1). Find S∞.
1. a₁ = 8, r = −0.5
2. |r| = 0.5 < 1 → converges
3. S∞ = 8 / (1−(−0.5)) = 8/1.5 = 16/3 ≈ 5.33
Worked Example — Find n
Q: Arithmetic seq, a₁=3, d=4. Which term equals 87?
1. aₙ = 3 + (n−1)·4 = 87
2. (n−1)·4 = 84 → n−1 = 21 → n = 22
Tip

If asked for "sum of terms from term 5 to term 10": compute S₁₀ − S₄. The −1 offset on the start matters.

Unit 2Transformations

General Form

y = a · f( b(x − h) ) + k Mapping rule: (x, y) → (x/b + h, a·y + k)
ParameterEffectTrap
aVertical stretch by |a|; a<0 reflects over x-axis|a|<1 = compression, not stretch
bHorizontal stretch by 1/|b|; b<0 reflects over y-axisb=2 means compress by ½, NOT stretch by 2
hShifts RIGHT by h (for positive h)y=f(x−3) shifts RIGHT 3, NOT left
kShifts UP by kNothing tricky here
h direction — most common error

y = f(x − h) → shifts RIGHT. The h inside the bracket is subtracted. Positive h = right.

y = f(x − 3): h = 3, shifts 3 right. y = f(x + 3): h = −3, shifts 3 left.

Reflections

  • Over x-axis: multiply outside → y = −f(x)
  • Over y-axis: negate x inside → y = f(−x)
  • Over y = x: swap x and y (this is the inverse)

Inverse Functions

To find f⁻¹(x): swap x and y, then solve for y
  • Domain of f = Range of f⁻¹
  • Range of f = Domain of f⁻¹
  • Graph of f⁻¹ is reflection of f over the line y = x
Worked Example — Identify Transformations
Q: y = −2f(3(x+1)) − 4. State all transformations.
1. a = −2 → vertical stretch ×2, reflect over x-axis
2. b = 3 → horizontal compression to 1/3
3. h = −1 → shift LEFT 1
4. k = −4 → shift DOWN 4
Mapping: (x,y) → (x/3 − 1, −2y − 4)
Tip — Applying the mapping rule

Take key points from the original function. Apply (x/b+h, ay+k) to each. Mapping order: horizontal changes first (divide x by b, shift by h), then vertical (multiply y by a, shift by k).

Unit 3Polynomials

Factoring Decision Tree

  1. Always factor GCF first
  2. 2 terms: diff of squares or sum/diff of cubes
  3. 3 terms, leading coeff = 1: simple trinomial (find p,q)
  4. 3 terms, leading coeff ≠ 1: AC method
  5. 4 terms: factor by grouping
  6. Degree 4 with x² pattern: substitution (let u = x²)
  7. Degree 3+: rational root theorem + synthetic division

Special Products

a² − b² = (a − b)(a + b) a³ + b³ = (a + b)(a² − ab + b²) a³ − b³ = (a − b)(a² + ab + b²)
Cubes — middle sign trap

Sum of cubes (a³+b³): middle term in trinomial is MINUS (a²−ab+b²)

Diff of cubes (a³−b³): middle term is PLUS (a²+ab+b²)

Memory aid: SOAP — Same, Opposite, Always Positive

AC Method (Messy Trinomials)

ax² + bx + c: find p,q where p·q = a·c AND p+q = b Split middle: ax² + px + qx + c, then group
Worked Example — AC Method
Q: Factor 6x² − 13x + 6
1. ac = 6·6 = 36. Find p,q: p·q=36, p+q=−13 → p=−9, q=−4
2. 6x² − 9x − 4x + 6
3. 3x(2x − 3) − 2(2x − 3)
4. (3x − 2)(2x − 3)

Remainder & Factor Theorems

P(a) = remainder when P(x) ÷ (x − a) [Remainder Theorem] P(a) = 0 ↔ (x − a) is a factor [Factor Theorem] Rational roots: ±(factors of constant) / (factors of leading coeff)
Worked Example — Factor Cubic
Q: Factor P(x) = x³ − 5x² − 12x + 36
1. Try rational roots: ±1,±2,±3,±4,±6,±9,±12,±18,±36
2. P(2) = 8−20−24+36 = 0 ✓ → (x−2) is a factor
3. Synthetic division → x³−5x²−12x+36 = (x−2)(x²−3x−18)
4. x²−3x−18 = (x−6)(x+3)
5. P(x) = (x−2)(x−6)(x+3)

Polynomial Graph Characteristics

  • Degree n → max n x-intercepts, max n−1 turning points
  • Leading coeff +, even degree → both ends go UP (U shape)
  • Leading coeff −, even degree → both ends go DOWN
  • Leading coeff +, odd degree → left DOWN, right UP
  • Leading coeff −, odd degree → left UP, right DOWN
  • Odd multiplicity at root → graph CROSSES x-axis
  • Even multiplicity at root → graph BOUNCES off x-axis

Unit 4Exponents & Logs

Exponent Laws

xᵃ · xᵇ = xᵃ⁺ᵇ (product — add exponents) xᵃ / xᵇ = xᵃ⁻ᵇ (quotient — subtract) (xᵃ)ᵇ = xᵃᵇ (power of power — multiply) x⁰ = 1 (zero exponent) x⁻ⁿ = 1/xⁿ (negative — flip) x^(m/n) = ⁿ√(xᵐ) (rational — root of power)

Logarithms

log_b(A) = E ↔ bᴱ = A (definition: log = exponent) log_b(AB) = log_b(A) + log_b(B) (product rule) log_b(A/B) = log_b(A) − log_b(B) (quotient rule) log_b(Aⁿ) = n · log_b(A) (power rule) log_b(x) = log(x) / log(b) (change of base)
Log = "what exponent do I need?"

log₂(8) = 3 because 2³ = 8. The log is the exponent. B = base, E = exponent, A = answer (result).

Solving Exponential Equations

  • Same base: 2^(x+1) = 2^5 → x+1 = 5 → x = 4
  • Different base: take log both sides, use power rule
3^x = 10 → x·log3 = log10 → x = log10/log3 ≈ 2.096

Growth & Decay

A = A₀ · b^(t/p) [b = growth factor, p = period for one cycle] A = A₀ · (1/2)^(t/h) [half-life: h = half-life period]
Known Error — U4 Q8 Half-life

When computing h = 20·log(0.5)/log(5/12): use exact fractions, don't round intermediate steps.

log(0.5) = −0.30103, log(5/12) = log(0.41667) = −0.37967

h = 20 · (−0.30103)/(−0.37967) = 15.84 ≈ 15.8 years

Log Scale Applications

  • Richter: difference in readings = exponent of 10× power. Δ=3.8 → 10³·⁸≈6310× more powerful
  • pH: pH = −log[H⁺]; more acidic = lower pH; ratio of acidity = 10^(pH₂−pH₁)
  • Decibels: dB = 10·log(I/I₀); increase of 10 dB = 10× louder
Worked Example — Change of Base
Q: Solve 5^x = 500
1. log both sides: x·log5 = log500
2. x = log500/log5
3. x = 2.699/0.699 ≈ 3.86
Worked Example — Log Rules Combined
Q: Write log x + 5 log y − 3 log x as single log
1. Combine x terms: (1−3)logx = −2logx
2. Power rule: log(x⁻²) + log(y⁵)
3. Product rule: log(y⁵/x²)

Unit 5Rational Expressions

Fundamentals

A rational expression is polynomial / polynomial. The denominator cannot equal zero.

Non-permissible values: set denominator = 0 and solve → those are EXCLUDED

Operations

OperationMethod
SimplifyFactor top and bottom fully, cancel common factors, state restrictions
MultiplyFactor everything, cancel across numerators/denominators, multiply remaining
DivideMultiply by reciprocal of second fraction, then same as multiply
Add/SubtractFind LCD (LCM of denominators), convert each fraction, combine numerators, simplify
Restrictions after cancelling

State ALL non-permissible values from the ORIGINAL denominators — even if a factor was cancelled. Restrictions come from before simplification.

Graphing y = a/(x−h) + k

Vertical asymptote: x = h (denominator = 0) Horizontal asymptote: y = k (what y approaches as x → ±∞) a: steepness; a > 0 → Q1/Q3 relative to asymptotes; a < 0 → Q2/Q4
Cheat code — finding 'a' from a graph

Pick any obvious grid point on the graph. Sub its (x,y) into y = a/(x−h)+k and solve for a. Always works.

Solving Rational Equations

  1. Identify all non-permissible values first
  2. Multiply every term by the LCD to clear fractions
  3. Solve the resulting polynomial equation
  4. Check solutions against non-permissible values — reject any that match
Worked Example — Find Equation from Graph
Q: Graph has VA at x=3, HA at y=2, passes through (4,7)
1. h=3, k=2. Form: y = a/(x−3) + 2
2. Sub (4,7): 7 = a/(4−3) + 2 = a + 2
3. a = 5 → y = 5/(x−3) + 2

Unit 6Sinusoidal Functions

General Form

y = a · sin( b(x − h) ) + k y = a · cos( b(x − h) ) + k
ParameterWhat it doesHow to find from graph
aAmplitude = |a|; a<0 flips verticallya = (max − min) / 2
bPeriod = 2π/|b| (in radians) or 360°/|b| (degrees)b = 2π / period
hPhase shift: moves graph right by hRead from graph where cycle starts
kVertical displacement (midline = y=k)k = (max + min) / 2

Graphing — 5 Key Points Method

For one full period of y = a·sin(b(x−h))+k:

  • Start at phase shift h: y = k (zero)
  • Quarter period: y = k+a (max)
  • Half period: y = k (zero)
  • Three-quarter period: y = k−a (min)
  • Full period: y = k (zero, cycle repeats)

For cosine: starts at max (k+a), not zero.

Worked Example — Equation from Graph
Q: Graph has max=7, min=1, period=4π, starts at x=π
1. a = (7−1)/2 = 3
2. k = (7+1)/2 = 4
3. b = 2π/4π = 1/2
4. h = π (phase shift right)
5. y = 3·sin(½(x−π)) + 4
Radians vs Degrees

Standard PC12 uses radians. Period formula: T = 2π/|b|. If period is given as "360°" then in radians that is 2π. Always match units.

Phase shift direction

y = sin(x − π/3): phase shift = +π/3 (shifts RIGHT). Same h-direction trap as U2 transformations.

Unit 7Trig Identities

Foundational Identities

Reciprocal: csc θ = 1/sin θ sec θ = 1/cos θ cot θ = 1/tan θ Quotient: tan θ = sin θ/cos θ cot θ = cos θ/sin θ Pythagorean: sin²θ + cos²θ = 1 Derived: tan²θ + 1 = sec²θ 1 + cot²θ = csc²θ
Deriving Pythagorean identities

Start from sin²θ+cos²θ=1. Divide both sides by cos²θ → tan²θ+1=sec²θ. Divide by sin²θ → 1+cot²θ=csc²θ. Don't memorize three — derive from one.

Proving Identities — Strategy

  1. Work ONE side only — never move terms across the = sign
  2. Pick the more complex side to transform
  3. Common techniques (in order to try):
  • Rewrite everything in terms of sin and cos
  • Factor out common terms
  • Get a common denominator to combine fractions
  • Multiply top and bottom by conjugate (1+sinθ, 1−cosθ, etc.)
  • Use a Pythagorean identity to substitute (sin²=1−cos² etc.)
Never cross-multiply to "prove"

Moving terms across the = sign assumes what you're trying to prove. Work one side only.

Sum & Difference Formulas

sin(A ± B) = sinA·cosB ± cosA·sinB cos(A ± B) = cosA·cosB ∓ sinA·sinB (note: ∓ flips the sign) tan(A ± B) = (tanA ± tanB) / (1 ∓ tanA·tanB)

Double Angle Formulas

sin(2A) = 2·sinA·cosA cos(2A) = cos²A − sin²A = 1 − 2sin²A = 2cos²A − 1 tan(2A) = 2·tanA / (1 − tan²A)
Three forms of cos(2A)

All three are equivalent via sin²+cos²=1. Pick whichever form makes the target expression cleanest. If the target has only cos, use 2cos²A−1.

Solving Trig Equations

  1. Isolate the trig function (treat sinθ or cosθ like a variable)
  2. Find the reference angle using inverse trig
  3. Identify which quadrants give the correct sign
  4. List all solutions in the given domain
  5. General solution: add 2πn (for sin/cos period 2π) or πn (for tan period π)
Worked Example — Solving with Identity
Q: Solve 2sin²θ − sinθ − 1 = 0, θ ∈ [0, 2π]
1. Factor: (2sinθ + 1)(sinθ − 1) = 0
2. sinθ = −½ → θ = 7π/6, 11π/6
3. sinθ = 1 → θ = π/2
Solutions: θ = π/2, 7π/6, 11π/6
